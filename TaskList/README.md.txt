# TaskList

### Description
An application that allows you to track and manage multiple tasks or goals.

Example:
> Enter a list of tasks you need to complete for the week and take notes on them and assign priority and track your completed tasks.

---

### Features
- Manage priority, due dates, and statuses.
- Tracks your tasks based on status, so you know how many tasks you have completed.
- Easy note feature that allows you to take notes per task and edit them as needed with timestamps.
- Great productivity tool for both home and work!
